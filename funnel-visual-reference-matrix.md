# Funnel Visual Reference Matrix

This pass is meant to stop the funnel exhibits from collapsing into the same visual family with different tints.

Typography guidance used here:
- Use one primary typeface family unless context genuinely changes.
- Introduce a secondary type treatment only when the context changes enough to justify it.
- Prefer clear hierarchy from size, weight, spacing, and rhythm before adding more visual decoration.
- When two type systems are needed, make them distinct enough to separate context but close enough to feel related.

Reference categories:

## VSL Funnel
- Real-world analog: agency or consultancy marketing page
- Typography: one strong sans system with large headline and restrained support copy
- Layout behavior: hero, proof row, guided overview, direct booking card
- Tone: confident, commercial, not theatrical

## DM / Conversation Funnel
- Real-world analog: CRM conversation handoff / inbound sales workspace
- Typography: compact sans, small status labels, clean message rail
- Layout behavior: conversation transcript, handoff summary, next action panel
- Tone: operational, immediate, contextual

## Qualification Funnel
- Real-world analog: intake portal / enterprise assessment form
- Typography: dense sans hierarchy, small labels, process framing, minimal flourish
- Layout behavior: staged form, fit criteria, reviewer sidebar, acceptance gate
- Tone: selective, procedural, serious

## Webinar Funnel
- Real-world analog: live workshop registration page
- Typography: bold event headline with a clean supporting sans system
- Layout behavior: event promise, date/time facts, agenda, registration card
- Tone: event-led, credible, polished

## Advertorial Funnel
- Real-world analog: consultancy thought-leadership article with inline conversion
- Typography: serif headline for long-form context, sans for supporting meta and CTA surfaces
- Layout behavior: article narrative, mechanism sections, side application panel
- Tone: explanatory, trust-building, editorial without cosplay

## Pre-Call Funnel
- Real-world analog: client prep portal / appointment center
- Typography: clean sans with dashboard-like labels and calm schedule detail
- Layout behavior: upcoming appointment, prep video, checklist, reminders, agenda
- Tone: dependable, structured, service-oriented

## Confirmation Funnel
- Real-world analog: appointment confirmation / service booking receipt
- Typography: practical sans with strong confirmation hierarchy
- Layout behavior: booking details, next steps, calendar notes, reminders
- Tone: reassuring, transactional, organized
